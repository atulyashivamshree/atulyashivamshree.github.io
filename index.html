Projects

Distance query with GPUs

Computing minimum distance between two objects is a critical task for safe navigation of autonomous vehicles. The task however can be computationally expensive involving large number of collision checks. In this project I parallelized this task for running on a GPU in order to run it faster. github page

Localization of underwater vehicles

Localization of underwater vehicles is a challenging task owing to lack of GPS signals or any electromagnetic-wave based communication. This project focuses on fusing acoustic measurements with onboard sensors for underwater localization of the vehicle. paper link

Vision based navigation of Quadcopter

This project implements waypoint navigation for quadcopters in indoor environments. It uses Open source monocular SLAM algorithms for localization along with sonar and inertial sensors. I worked on integrating the visual odometry pipeline for the state estimation. A cascaded loop PID controller is used for position control.

Chess game recognition from images

Imagine if your phone could capture all the moves of a board game. You could use it in an AR pipeline to play with a virtual player. Or you could collect statistics and analyze your game at a later time. This project implements move detection for a game of chess from a handheld camera. It uses AR markers for tracking the chess board so that you can record the game from any angle.
